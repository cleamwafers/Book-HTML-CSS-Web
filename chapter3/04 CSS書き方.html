書き方〜その１〜

h1{
color: red;
}

書き方〜その2〜

h1{ color: red; }

名前

名前 用途
セレクター どの部分を装飾するのかの指定
プロパティ セレクターで指定された部分の何を変えるのかを書く。
例：文字色、文字サイズ、背景画像
値 見た目をどのように変えるのか
例：文字色を何色にするか、どの画像を使うのか

ルール

方法 例
半角英数字で書く h1 { color: red;}
基本小文字で記述する h1 { color: red;}
複数のセクレターに指定をするときはカンマで区切る h1,p { color: red;}
複数の装飾を指定をするときはセミコロンで区切る h1{ color: red; font-size: 20px; }

単位を指定する

単位 読み方 説明 例
px(絶対値：指定したサイズから変更はない) ピクセル 画面上の最小単位（1ピクセル）を基準とした単位 h1 { font-size: 20px;}
% パーセント 親要素のサイズを基準に割合で指定する単位
rm（相対値：基準となる要素のサイズによって変動する） レム ルート要素(html要素)に指定されたサイズを基準とした単位
